# Release `v0.1.0`

## Release

- version: `v0.1.0`
- date: `2026-03-09`

## Summary

- establishes the L0 coordinated image set for platform, relay, and operator surfaces
- defines the compose-first self-hosted deployment entrypoint
- keeps npm packages available for build and validation, but not as the main operator installation path

## Included Images

- `rsp-platform:v0.1.0`
- `rsp-gateway:v0.1.0`
- `rsp-relay:v0.1.0`

## Validation

- platform integration checks
- deploy config validation
- service package smoke validation

## Readiness Boundary

- `v0.1.0` is a coordinated pilot baseline
- it is not the final production-ready promise

## Notes

- operator entry is Docker and compose
- mixed-version deployments are not part of the L0 support promise
